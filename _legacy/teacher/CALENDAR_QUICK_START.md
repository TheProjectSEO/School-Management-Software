# Calendar Quick Start Guide

## Accessing the Calendar

Navigate to: `/teacher/calendar`

## Quick Actions

### Create a Live Session
1. Click **"Create Session"** button (top right)
2. Fill in the form:
   - **Subject/Course**: Required - Select from your assigned courses
   - **Module**: Optional - Link to specific module
   - **Session Title**: Required - e.g., "Introduction to Algebra"
   - **Description**: Optional - Session details
   - **Start Date/Time**: Required - When session begins
   - **End Date/Time**: Optional - When session ends
   - **Video Platform**: Choose provider (Zoom, Meet, Teams, etc.)
   - **Join URL**: Meeting link for students
3. Click **"Create Session"**

### View Session Details
- Click any session event on the calendar
- Detail panel slides in from right
- Shows full session information
- Access join URL and recording (if available)

### Edit a Session
1. Click session to open details panel
2. Click **"Edit Session"** button
3. Modify fields as needed
4. Save changes

### Delete a Session
1. Click session to open details panel
2. Click **"Delete Session"** button
3. Confirm deletion
4. Session removed immediately

## Calendar Navigation

### View Modes
- **Month View**: See entire month at a glance
- **Week View**: Hourly schedule for 7 days
- **Day View**: Detailed single-day schedule

Switch using tabs at top of calendar.

### Time Navigation
- **Previous (◀)**: Go back one month/week/day
- **Next (▶)**: Go forward one month/week/day
- **Today**: Jump to current date

## Understanding Status Colors

### Live Sessions
- **Blue**: Scheduled (upcoming)
- **Green**: Live (currently happening)
- **Gray**: Ended (completed)
- **Red**: Cancelled

### Assessments
- **Yellow/Orange**: Due dates

## Tips & Tricks

### Best Practices
1. **Add Join URLs Early**: Students need meeting links before session
2. **Set End Times**: Helps with scheduling conflicts
3. **Link to Modules**: Better organization and tracking
4. **Use Descriptions**: Help students prepare for sessions

### Keyboard Shortcuts
- **ESC**: Close detail panel or modal
- **Click Outside**: Dismiss panels

### Mobile Usage
- Calendar adapts to screen size
- Swipe to navigate dates
- Tap events for details
- Full functionality on all devices

## Common Workflows

### Weekly Planning
1. Switch to **Week View**
2. Click **"Create Session"** for each class
3. Fill in details and save
4. Review week layout

### Monthly Overview
1. Switch to **Month View**
2. Check for scheduling conflicts
3. Note assessment due dates
4. Plan ahead for busy periods

### Day-of Session
1. Switch to **Day View**
2. Click session to open details
3. Copy join URL to share with students
4. Mark as "Live" when starting (future feature)

## Troubleshooting

### "No sessions to display"
- You haven't created any sessions yet
- Sessions are outside current view range
- Try clicking "Today" button

### Can't create session
- Check that you're assigned to the course
- All required fields must be filled
- Check internet connection

### Session doesn't appear
- Wait for page to refresh
- Try refreshing browser (F5)
- Check selected date range
