================================================================================
              MSU SCHOOL MANAGEMENT SYSTEM - USER CAPABILITIES
================================================================================

================================================================================
  ADMIN CAPABILITIES
================================================================================

  DASHBOARD & ANALYTICS
  ---------------------
  - View KPI cards (total students, teachers, courses, enrollments, attendance)
  - Enrollment trends chart (monthly)
  - Grade distribution chart (A-F breakdown)
  - Attendance overview (present/late/absent/excused)
  - Recent activity audit log
  - Churn prediction analytics (at-risk students)
  - Quick actions (import students, add teacher, bulk enroll)

  USER MANAGEMENT - STUDENTS
  --------------------------
  - View all students (paginated, search, filter by status/grade/section)
  - View student details (profile, enrollment history, attendance, grades)
  - Create student account
  - Edit student info (name, email, LRN, grade level, section)
  - Change student status (active/inactive/suspended)
  - Reset student password
  - Bulk import students (CSV upload)
  - Bulk update student status
  - Bulk assign students to section (auto-enrolls to section courses)
  - Export student list (CSV/Excel)

  USER MANAGEMENT - TEACHERS
  --------------------------
  - View all teachers (paginated, search, filters)
  - View teacher details (profile, assigned courses, sections, students)
  - Create teacher account
  - Edit teacher info (name, email, specialization)
  - Assign teacher to courses
  - Assign advisory section
  - Change teacher status (active/inactive/suspended)
  - Reset teacher password
  - Bulk import teachers (CSV upload)
  - Bulk update teacher status
  - Export teacher list (CSV/Excel)

  ENROLLMENT MANAGEMENT
  ---------------------
  - View all enrollments (filtered list)
  - Create enrollment (manually enroll student in course)
  - Bulk enroll students to courses
  - Approve pending enrollment requests
  - Drop enrollment (remove student from course)
  - Transfer enrollment (move student to different section/course)
  - Edit enrollment details
  - Delete enrollment record
  - Export enrollments (CSV)
  - Sync section enrollments (backfill missing records)
  - QR code enrollment check-in

  COURSES & SECTIONS
  ------------------
  - View/create/edit courses (name, code, description, grade level)
  - Bulk add subjects from template
  - View/create/edit sections (grade/class groupings)
  - Assign courses to sections
  - View section-course assignments

  REPORTS & ANALYTICS
  -------------------
  - Attendance report (by student, date range, status)
  - Grades report (distribution, per-student, per-assessment)
  - Progress report (module completion rates, learning progress)
  - Churn analysis (dropout predictions, at-risk identification)
  - Export reports

  FINANCE
  -------
  - Create/edit/delete fee categories (tuition, books, uniforms, etc.)
  - Create/edit/delete fee structures (per grade level/section)
  - Assess fees to students (bulk assign for period)
  - View/edit student financial accounts
  - Create/edit/delete payment plans
  - Record manual payments
  - Process online payments (PayMongo)
  - View overdue accounts and collection status

  APPLICATIONS & ADMISSIONS
  -------------------------
  - View all applications
  - View application details and documents
  - AI-screen applications (single or bulk)
  - Approve/reject applications
  - Request additional info from applicants

  SETTINGS & CONFIGURATION
  ------------------------
  - School settings (name, logo, contact info, calendar)
  - Academic settings (academic year, grading scale, grade levels)
  - Grading periods (create/edit quarters)
  - Academic years (create, mark active)

  MESSAGING
  ---------
  - View all conversations
  - Send direct messages to any user
  - Search message history

  AUDIT & COMPLIANCE
  ------------------
  - View audit logs (all system actions)
  - Filter logs by action, user, entity, date
  - View log details (who did what, when)

  INQUIRIES
  ---------
  - View and respond to chatbot inquiries from applicants

  ANNOUNCEMENTS

  - Create/edit/delete announcements
